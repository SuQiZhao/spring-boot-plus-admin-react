# spring-boot-plus-admin-react

## 管理系统主页
[http://47.105.159.10/](http://47.105.159.10/)

### 设置nodejs镜像
```text
npm config set registry https://registry.npm.taobao.org
```

### npm install
```text
npm install
```

## 使用方式一
### npm启动应用
```text
npm start
```

### npm build
> 打包后的目录：dist
```text
npm run-script build
```

## 使用方式二
### 安装umi
```text
sudo npm install -g umi
```

### 启动应用
```text
umi dev
```

### 打包
```text
umi build
```
